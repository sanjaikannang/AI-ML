List

→ list is an ordered collection of items that can be of any type (integers, strings, objects, etc.). Lists are mutable, meaning you can modify them after they are created. Lists are defined using square brackets [], and the elements are separated by commas.

Basic Example:

# Defining a list
my_list = [1, 2, 3, 4, 5]
print(my_list)  

Output: 

[1, 2, 3, 4, 5]

Accessing List Element

You can access individual elements of a list using indices (0-based index):

my_list = ['apple', 'banana', 'cherry']
print(my_list[0])  # Output: apple
print(my_list[1])  # Output: banana
print(my_list[2])  # Output: cherry

Modifying Lists:

Since lists are mutable, you can modify elements.

my_list = [10, 20, 30]
my_list[1] = 25  # Change the second element
print(my_list)  

Output: 

[10, 25, 30]

Adding Elements to a List:

You can add elements using append() or insert():

# Append adds to the end of the list
my_list = [1, 2, 3]
my_list.append(4)
print(my_list)  

Output: 

[1, 2, 3, 4]


# Insert adds at a specific position
my_list.insert(1, 'apple')
print(my_list)  

Output: 

[1, 'apple', 2, 3, 4]

Removing Elements from a List:

You can remove elements using remove(), pop(), or del:

my_list = ['a', 'b', 'c', 'd']

# Using remove (removes by value)
my_list.remove('b')
print(my_list)  

Output: 

['a', 'c', 'd']


# Using pop (removes by index, default is last item)
my_list.pop()
print(my_list)  

Output: 

['a', 'c']

# Using del (removes by index)
del my_list[0]
print(my_list)  

Output: 

['c']

Other Useful List Operations:

  1. List Length: Get the number of elements.
    my_list = [1, 2, 3]
print(len(my_list))

Output: 

3
  1. List Slicing: Get a part of the list.
    my_list = [10, 20, 30, 40, 50]
print(my_list[1:4])  

Output: 

[20, 30, 40]
  1. List Concatenation: Combine two lists.
    list1 = [1, 2]
list2 = [3, 4]
print(list1 + list2)  

Output: 

[1, 2, 3, 4]
  1. List Repetition: Repeat elements in the list.
    list1 = [1, 2]
print(list1 * 3)  

Output: 

[1, 2, 1, 2, 1, 2]

Nested Lists:

You can have lists inside other lists (nested lists).

nested_list = [[1, 2], [3, 4], [5, 6]]
print(nested_list[0])  

Output: 

[1, 2]

print(nested_list[1][1])  

Output: 

4

List Comprehension:

List comprehension provides a concise way to create lists.

# Creating a list of squares of numbers from 0 to 4
squares = [x ** 2 for x in range(5)]
print(squares)  

Output: 

[0, 1, 4, 9, 16]

Set

→ A set is an unordered collection of unique elements. Sets are useful when you want to store multiple values but don't care about their order and need to ensure that no duplicate values are present. A set is defined using curly braces {} or the set() function.

Basic Example:

# Defining a set
my_set = {1, 2, 3, 4, 5}
print(my_set)  

Output: 

{1, 2, 3, 4, 5}

Adding Elements to a Set:

You can add elements using the add() method. If you try to add a duplicate element, it will not be added since sets don’t allow duplicates.

my_set = {1, 2, 3}
my_set.add(4)  # Adds 4 to the set
print(my_set)  # Output: {1, 2, 3, 4}

# Adding a duplicate element (won't be added)
my_set.add(2)
print(my_set)  

Output: 

{1, 2, 3, 4} (no change)

Removing Elements from a Set:

You can remove elements using remove() or discard(). The difference is that remove() will raise a KeyError if the element is not found, while discard() will not.

my_set = {1, 2, 3, 4}

# Using remove (raises KeyError if element is not found)
my_set.remove(3)
print(my_set)  

Output: {1, 2, 4}

# Using discard (does not raise KeyError)
my_set.discard(5)  # No effect, 5 is not in the set
print(my_set)  

Output: 

{1, 2, 4}

Set Operations:

  1. Union: Combines two sets, removing duplicates.
    set1 = {1, 2, 3}
set2 = {3, 4, 5}
union_set = set1 | set2  # Or set1.union(set2)
print(union_set)  

Output: 

{1, 2, 3, 4, 5}
  1. Intersection: Returns common elements between two sets.
    set1 = {1, 2, 3}
set2 = {3, 4, 5}
intersection_set = set1 & set2  # Or set1.intersection(set2)
print(intersection_set)  

Output: 

{3}
  1. Difference: Returns elements that are in the first set but not in the second.
    set1 = {1, 2, 3}
set2 = {3, 4, 5}
difference_set = set1 - set2  # Or set1.difference(set2)
print(difference_set)  

Output: 

{1, 2}
  1. Symmetric Difference: Returns elements that are in either of the sets but not in both.
    set1 = {1, 2, 3}
set2 = {3, 4, 5}
symmetric_difference_set = set1 ^ set2  # Or set1.symmetric_difference(set2)
print(symmetric_difference_set)  

Output: 

{1, 2, 4, 5}

Set Comprehension:

You can create sets in a concise way using set comprehension.

# Creating a set of squares of numbers from 0 to 4
squares_set = {x ** 2 for x in range(5)}
print(squares_set)  

Output: 

{0, 1, 4, 9, 16}

Checking Membership:

You can check if an element exists in a set using the in keyword.

my_set = {1, 2, 3, 4}

print(3 in my_set)  # Output: True
print(5 in my_set)  # Output: False

Set Length:

You can find the number of elements in a set using len().

my_set = {10, 20, 30}
print(len(my_set))  

Output: 

3

Set Example with Mixed Data Types:

Sets can also store mixed data types like strings, integers, and other objects.

my_set = {1, "apple", 3.14, (1, 2)}
print(my_set)  

Output: 

{1, 'apple', 3.14, (1, 2)}

Set Comparison:

You can compare sets for equality, subset, or superset relationships:

set1 = {1, 2, 3}
set2 = {3, 2, 1}
set3 = {4, 5, 6}

print(set1 == set2)  

Output: 

True (since sets are unordered)

print(set1 < set3)   

Output: 

False (set1 is not a subset of set3)

Set Discarding Duplicates:

You can use a set to automatically remove duplicates from a list:

list_with_duplicates = [1, 2, 2, 3, 4, 4, 5]
unique_elements = set(list_with_duplicates)
print(unique_elements)  

Output: 

{1, 2, 3, 4, 5}

Tuple

→ A tuple is an ordered collection of elements that is immutable, meaning that once a tuple is created, its elements cannot be changed, added, or removed. Tuples are defined using parentheses ().

Basic Example:

# Defining a tuple
my_tuple = (1, 2, 3, 4, 5)
print(my_tuple)  

Output: 

(1, 2, 3, 4, 5)

Accessing Tuple Elements:

You can access individual elements of a tuple using indices (0-based index):

my_tuple = ('apple', 'banana', 'cherry')
print(my_tuple[0])  # Output: apple
print(my_tuple[1])  # Output: banana
print(my_tuple[2])  # Output: cherry

Slicing Tuples:

You can access a range of elements using slicing.

my_tuple = (10, 20, 30, 40, 50)
print(my_tuple[1:4])

Output: 

(20, 30, 40)

Nested Tuples:

Tuples can contain other tuples as elements (nested tuples).

nested_tuple = (1, 2, (3, 4), 5)
print(nested_tuple[2])  # Output: (3, 4)
print(nested_tuple[2][1])  # Output: 4

Tuple Concatenation and Repetition:

  1. Concatenation: Combine two or more tuples.
    tuple1 = (1, 2)
tuple2 = (3, 4)
concatenated_tuple = tuple1 + tuple2
print(concatenated_tuple)  

Output: 

(1, 2, 3, 4)
  1. Repetition: Repeat the elements of a tuple.
    tuple1 = (1, 2)
repeated_tuple = tuple1 * 3
print(repeated_tuple)  

Output: 

(1, 2, 1, 2, 1, 2)

Tuple Length:

You can find the number of elements in a tuple using len().

my_tuple = (10, 20, 30)
print(len(my_tuple))   

Output: 

3

Tuple Membership:

You can check if an element exists in a tuple using the in keyword.

my_tuple = (1, 2, 3, 4)
print(3 in my_tuple)  # Output: True
print(5 in my_tuple)  # Output: False

Unpacking Tuples:

You can unpack a tuple into individual variables.

my_tuple = (1, 2, 3)
a, b, c = my_tuple
print(a)  # Output: 1
print(b)  # Output: 2
print(c)  # Output: 3

Tuple with Different Data Types:

A tuple can contain elements of different data types such as integers, strings, and even other tuples.

my_tuple = (1, 'apple', 3.14, (2, 3))
print(my_tuple)  

Output: 

(1, 'apple', 3.14, (2, 3))

One-Element Tuple:

To create a tuple with just one element, you need to include a trailing comma.

single_element_tuple = (5,)
print(single_element_tuple)  

Output: 

(5)

Tuple Indexing and Negative Indexing:

You can access elements using both positive and negative indices. Negative indexing starts from the end of the tuple.

my_tuple = ('a', 'b', 'c', 'd', 'e')
print(my_tuple[1])  # Output: b
print(my_tuple[-1])  # Output: e

Tuple Methods:

  1. count(): Returns the number of occurrences of an element in the tuple.
    my_tuple = (1, 2, 2, 3, 4, 2)
print(my_tuple.count(2))  

Output: 

3
  1. index(): Returns the index of the first occurrence of an element.
    my_tuple = (1, 2, 3, 4, 5)
print(my_tuple.index(3))  

Output: 

2

Dictionary


→ A
dictionary is an unordered collection of key-value pairs. Each key in the dictionary is unique, and values can be of any data type. Dictionaries are defined using curly braces {} with the key-value pairs separated by a colon :.

Basic Example:

# Defining a dictionary
my_dict = {"name": "Alice", "age": 25, "city": "New York"}
print(my_dict)  

Output: 

{'name': 'Alice', 'age': 25, 'city': 'New York'}

Accessing Values by Key:

You can access the value associated with a specific key using square brackets [].

my_dict = {"name": "Alice", "age": 25, "city": "New York"}

print(my_dict["name"])  # Output: Alice
print(my_dict["age"])   # Output: 25

Modifying Dictionary Values:

Since dictionaries are mutable, you can change the value of an existing key.

my_dict = {"name": "Alice", "age": 25}
my_dict["age"] = 26  # Modify the value of 'age'
print(my_dict)  

Output: 

{'name': 'Alice', 'age': 26}

Adding Key-Value Pairs:

You can add new key-value pairs to a dictionary.

my_dict = {"name": "Alice", "age": 25}
my_dict["city"] = "New York"  # Add a new key-value pair
print(my_dict)  

Output: 

{'name': 'Alice', 'age': 25, 'city': 'New York'}

Removing Key-Value Pairs:

You can remove key-value pairs using del, pop(), or popitem().

my_dict = {"name": "Alice", "age": 25, "city": "New York"}

# Using del (removes the key-value pair by key)
del my_dict["city"]
print(my_dict)  

Output: 

{'name': 'Alice', 'age': 25}


# Using pop (removes the key-value pair and returns the value)
removed_value = my_dict.pop("age")
print(removed_value)  # Output: 25
print(my_dict)  # Output: {'name': 'Alice'}

# Using popitem (removes and returns an arbitrary key-value pair)
item = my_dict.popitem()
print(item)  # Output: ('name', 'Alice')
print(my_dict)  # Output: {}

Dictionary Keys, Values, and Items:

  1. Keys: You can get a list of all the keys in the dictionary.
    my_dict = {"name": "Alice", "age": 25, "city": "New York"}
print(my_dict.keys())  

Output: 

dict_keys(['name', 'age', 'city'])
  1. Values: You can get a list of all the values in the dictionary.
    print(my_dict.values())  

Output: 

dict_values(['Alice', 25, 'New York'])
  1. Items: You can get a list of all the key-value pairs in the dictionary.
    print(my_dict.items())  

Output: dict_items([('name', 'Alice'), ('age', 25), ('city', 'New York')])

Checking if a Key Exists:

You can check if a specific key exists in the dictionary using the in keyword.

my_dict = {"name": "Alice", "age": 25}
print("name" in my_dict)  # Output: True
print("address" in my_dict)  # Output: False

Iterating Through a Dictionary:

You can loop through the keys, values, or key-value pairs of a dictionary.

my_dict = {"name": "Alice", "age": 25, "city": "New York"}

# Iterating through keys
for key in my_dict:
    print(key)  
    
Output: 

name, age, city


# Iterating through values
for value in my_dict.values():
    print(value)  
    
Output: 

Alice, 25, New York


# Iterating through key-value pairs
for key, value in my_dict.items():
    print(key, ":", value)
    
    
Output: 
name : Alice
age : 25
city : New York

Dictionary Comprehension:

You can create dictionaries using a concise syntax called dictionary comprehension.

# Create a dictionary with squares of numbers
squares = {x: x**2 for x in range(5)}
print(squares)  

Output: 

{0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

Nested Dictionaries:

Dictionaries can contain other dictionaries as values (nested dictionaries).

nested_dict = {
    "person1": {"name": "Alice", "age": 25},
    "person2": {"name": "Bob", "age": 30}
}
print(nested_dict["person1"]["name"])  # Output: Alice
print(nested_dict["person2"]["age"])   # Output: 30

Copying a Dictionary:

You can create a shallow copy of a dictionary using the copy() method.

my_dict = {"name": "Alice", "age": 25}
new_dict = my_dict.copy()
print(new_dict)  

Output: 

{'name': 'Alice', 'age': 25}

Dictionary Default Values with get():

The get() method is used to access a value by its key and provides a default value if the key doesn’t exist.

my_dict = {"name": "Alice", "age": 25}
print(my_dict.get("name"))  # Output: Alice
print(my_dict.get("address", "Not Available"))  

Output: 

Not Available
