Conditions

  • Equals: a == b
  • Not Equals: a != b
  • Less than: a < b
  • Less than or equal to: a <= b
  • Greater than: a > b
  • Greater than or equal to: a >= b

If Statement / Elif Statement / Else Statement

1. If Statement

The if statement is used to execute a block of code only if a certain condition is True.

Example:

x = 10
if x > 5:
    print("x is greater than 5")

OutPut: 

x is greater than 5

2. Elif Statement

The elif (short for "else if") is used when there are multiple conditions to check. It follows an if statement and can check additional conditions if the initial if condition is not true.

Example:

x = 10
if x > 15:
    print("x is greater than 15")
elif x > 5:
    print("x is greater than 5 but less than or equal to 15")

OutPut:   
 
x is greater than 5 but less than or equal to 15

3. Else Statement

The else statement is used to specify a block of code that will run if none of the previous if or elif conditions were true.

Example:

x = 3
if x > 5:
    print("x is greater than 5")
else:
    print("x is not greater than 5")

OutPut:   
 
x is not greater than 5

If with and/ If with or / If with not

If with and

You can combine multiple conditions using the and logical operator. The code block will execute only if both conditions are true.

Example:

x = 10
y = 20
if x > 5 and y < 25:
    print("Both conditions are True")

OutPut:

Both conditions are True

If with or

You can use the or logical operator to check if at least one condition is true. The code block will execute if either condition is true.

Example:

x = 10
y = 30
if x > 5 or y < 25:
    print("At least one condition is True")

OutPut:

At least one condition is True

If with not

The not operator is used to invert a condition. The code block will execute if the condition is False.

Example:

x = 10
if not x > 15:
    print("x is not greater than 15")

OutPut:

x is not greater than 15

For Loop

A for loop is used to iterate over a sequence (like a list, tuple, string, or range). It allows you to execute a block of code multiple times.

Example:

for i in range(5):
    print(i)
    
OutPut:

0
1
2
3
4

Combining If Statements and For Loops

You can combine if statements with for loops to make decisions while iterating.

Example:

numbers = [1, 2, 3, 4, 5]
for number in numbers:
    if number % 2 == 0:
        print(f"{number} is even")
    else:
        print(f"{number} is odd")

Outut:

1 is odd
2 is even
3 is odd
4 is even
5 is odd
