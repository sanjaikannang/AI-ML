1. What is a Function?

A function in Python is a defined block of code that can be executed when called. It allows you to bundle code into a logical unit that you can use repeatedly.

Syntax of a Function:

def function_name(parameters):
    # Function body
    # Do something
    return value  # optional
  • def: This is the keyword that defines a function.
  • function_name: The name of the function.
  • parameters: These are the inputs you pass to the function (optional).
  • return: This is used to return a value from the function (optional).

Example:

def greet(name):
    print(f"Hello, {name}!")

greet("Alice")  # Output: Hello, Alice!

2. Types of Functions

Functions in Python can be categorized in different ways based on how they are defined and used:

A. Built-in Functions:

Python comes with many built-in functions that are available for use without needing to define them. Examples include print(), len(), int(), str(), sum(), etc.

print("Hello World!")  # print is a built-in function

B. User-Defined Functions:

These are functions that you define yourself using the def keyword, like the greet function example shown earlier.

def square(x):
    return x * x

C. Lambda Functions (Anonymous Functions):

Lambda functions are small anonymous functions defined with the lambda keyword. They are generally used when you need a simple function for a short period of time, usually for passing it as an argument.

multiply = lambda x, y: x * y
print(multiply(2, 3))  # Output: 6

D. Recursive Functions:

A recursive function is a function that calls itself in order to solve a problem. Typically, a recursive function has a base case to avoid infinite recursion.

def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

E. Higher-Order Functions:

Higher-order functions are functions that take other functions as arguments or return a function as a result. Python's built-in functions like map(), filter(), and reduce() are examples of higher-order functions.

def apply_function(func, value):
    return func(value)

result = apply_function(lambda x: x**2, 5)
print(result)  # Output: 25

3. Function Arguments

Functions can accept different types of arguments. Here’s a look at some of them:

A. Positional Arguments:

These are the most common kind of arguments. The arguments are passed in the order in which they are defined.

def add(a, b):
    return a + b

print(add(3, 5))  # Output: 8

B. Keyword Arguments:

You can also pass arguments by name, not just position. This improves clarity.

def greet(name, age):
    print(f"Hello {name}, you are {age} years old.")

greet(age=25, name="Bob")

C. Default Arguments:

You can assign default values to parameters. If a value is not provided during the function call, the default is used.

def greet(name, age=30):
    print(f"Hello {name}, you are {age} years old.")

greet("Alice")  # Uses the default value for age

D. Variable-length Arguments (Arbitrary Arguments):

Sometimes, you don’t know in advance how many arguments will be passed to your function. You can use *args and **kwargs for this.

  • args allows you to pass a variable number of non-keyword arguments (a tuple).
  • *kwargs allows you to pass a variable number of keyword arguments (a dictionary).
def sum_all(*args):
    return sum(args)

print(sum_all(1, 2, 3, 4))  # Output: 10

def greet_all(**kwargs):
    for name, age in kwargs.items():
        print(f"Hello {name}, you are {age} years old.")

greet_all(Alice=25, Bob=30)

4. Return Statement

The return statement is used to send the result of a function back to the caller. If no return statement is present, the function will return None by default.

def multiply(a, b):
    return a * b

result = multiply(3, 4)
print(result)  # Output: 12

If there’s no return, the function implicitly returns None:

def say_hello():
    print("Hello!")

print(say_hello())  # Output: None

5. Scope and Lifetime of Variables

Variables in functions have different scopes, meaning they exist only in certain contexts.

  • Local Variables: Variables defined inside a function are local to that function and can only be accessed within it.
  • Global Variables: Variables defined outside all functions are global and can be accessed inside any function (unless shadowed by a local variable).
x = 10  # global variable

def display():
    print(x)  # access global variable

display()  # Output: 10

If you want to modify a global variable inside a function, you must use the global keyword:

x = 10

def modify_global():
    global x
    x = 20

modify_global()
print(x)  # Output: 20

6. Docstrings

Functions can also have docstrings to describe what the function does. Docstrings are useful for documentation and can be accessed using the help() function.

def add(a, b):
    """This function returns the sum of a and b."""
    return a + b

print(add.__doc__)  # Output: This function returns the sum of a and b.

7. Function Decorators (Advanced Topic)

A function decorator is a special type of function that takes another function as input and modifies its behavior. This is often used to add additional functionality, like logging or authentication.

def decorator(func):
    def wrapper():
        print("Before function call")
        func()
        print("After function call")
    return wrapper

@decorator
def say_hello():
    print("Hello!")

say_hello()

Output:

Before function call
Hello!
After function call
