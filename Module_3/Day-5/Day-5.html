1. Lambda Functions

What is it?

A lambda function is a small anonymous function defined with the lambda keyword. It’s often used for short operations or simple functions that you don't want to define with the regular def syntax.

Syntax:

lambda arguments: expression

Example:

multiply = lambda x, y: x * y
print(multiply(2, 3))  # Output: 6

Why is it useful?

  • Lambda functions are useful when you need a function for a short period of time and don’t want to write a full function definition.
  • They’re great for functional programming, especially when used with map(), filter(), or sorted() functions.

Where to use it?

  • When you need a small, throwaway function that is only used in a specific context.
  • Commonly used as arguments in higher-order functions (e.g., map(), filter(), sorted()).

2. List Comprehension

What is it?

List comprehension provides a concise way to create lists in Python. It allows you to generate a list by applying an expression to each item in an iterable (e.g., a list or range).

Syntax:

[expression for item in iterable if condition]

Example:

squared_numbers = [x**2 for x in range(5)]  # Output: [0, 1, 4, 9, 16]

Why is it useful?

  • It's more compact and readable than using traditional loops to create lists.
  • It helps in avoiding verbose code, making your code more Pythonic.

Where to use it?

  • Whenever you want to create a new list from an existing iterable, applying a transformation to the items in the process.
  • When you need to filter items based on a condition and create a new list.

3. Filter

What is it?

The filter() function is used to filter elements from an iterable based on a function that tests each element. It returns a new iterable (filter object), containing only the items where the function returned True.

Syntax:

filter(function, iterable)

Example:

# Remove even numbers from a list
numbers = [1, 2, 3, 4, 5, 6]
even_numbers = filter(lambda x: x % 2 == 0, numbers)
print(list(even_numbers))  # Output: [2, 4, 6]

Why is it useful?

  • It's used to clean data or filter specific items from a collection based on a condition.
  • Helps write cleaner, more declarative code rather than using a loop to filter items manually.

Where to use it?

  • When you need to remove elements from an iterable based on a condition.
  • Commonly used for data processing tasks, such as removing invalid entries or filtering out specific types of data.

4. Map

What is it?

The map() function applies a given function to all items in an iterable (such as a list) and returns a map object (which can be converted to a list or other iterable).

Syntax:

map(function, iterable)

Example:

# Multiply each number by 2
numbers = [1, 2, 3, 4]
doubled_numbers = map(lambda x: x * 2, numbers)
print(list(doubled_numbers))  # Output: [2, 4, 6, 8]

Why is it useful?

  • map() provides a clean and functional way to apply transformations to each element in a collection.
  • It’s more concise and efficient than using a for loop for applying the same operation to all items in an iterable.

Where to use it?

  • When you need to apply a specific operation to every element in an iterable.
  • Ideal for situations where you're transforming data, such as converting all strings in a list to uppercase or doubling numbers.
